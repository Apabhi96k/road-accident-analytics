# Road Accident Analytics Platform

This project analyzes road accident data using Python and Data Science techniques.

## Features
- Weather impact analysis
- Vehicle risk analysis
- Driver age analysis
- Road type distribution
- Accident trend forecasting
- Accident risk prediction

## Technologies Used
Python  
Pandas  
NumPy  
Plotly  
Scikit-learn  
Streamlit

## How to Run

Install dependencies:

pip install -r requirements.txt

Run application:

streamlit run dashboard.py
